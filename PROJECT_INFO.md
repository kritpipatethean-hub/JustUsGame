# Easy Brain Game: 4-in-a-row Multiplayer

This file contains the context, requirements, and strict rules for this project.

## Project Rules for Agents
1. **Scope Restriction**: DO NOT delete, modify, or interact with files outside of this specific project folder (`c:/Users/EarthPad/Desktop/Website/Easy Brain Game`).
2. **Language Requirement**: The web application's user interface MUST be entirely in English.
3. **Multiplayer Architecture**: The game uses Firebase Realtime Database for online matchmaking and state synchronization.
4. **Target Audience**: This is a closed system for exactly two specific players: "Earth" and "Noon".
5. **Expandability**: The architecture must be built as a "Game Hub" to support adding multiple new games in the future easily.
6. **Statistics**: Win/Loss/Draw statistics must be tracked and displayed for all games.
7. **Version Control**: When the project is ready to be pushed to GitHub, it MUST be pushed to a **NEW, separate repository**. Do not mix it with other existing projects.
